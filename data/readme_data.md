# Argus AI: Regulatory Dataset

## Dataset Description
This folder contains the raw, unstructured regulatory policy documents utilized to evaluate the Argus AI compliance engine. The dataset spans three major compliance frameworks to test the Node 1 Triage Router and Node 2 Semantic Auditor:

1. NIST Framework (Federal Cybersecurity):
   - NIST.SP.800-53r5.pdf (Security and Privacy Controls for Information Systems)

2. GDPR Framework (European Data Privacy):
   - CELEX_32016R0679_EN_TXT.pdf (General Data Protection Regulation text)

3. HIPAA Framework (Healthcare Privacy):
   - consumer_rights.pdf
   - sharing-family-friends.pdf
   - privacy-security-electronic-records.pdf
   - ocr-letter-hipaa-privacy-rule-and-parental-access-to-minor-childrens-medical-records.pdf

These documents serve as the ground-truth input variables for the RAG Evidence Extractor and semantic entailment loops.
